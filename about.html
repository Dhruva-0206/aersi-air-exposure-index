About

What is AERSI?

AERSI is an independent research project building a better way to understand air pollution exposure in India — going beyond the daily AQI snapshot to capture how intense, persistent, and volatile the exposure truly is.

The Project

Why this exists

🎯 The Goal

To build an open, scientifically grounded index that tells the real story of air pollution exposure in India — not just a daily snapshot, but a rolling, multi-dimensional measure of how dangerous the air actually is over time.

🔬 The Approach

Three components — Pollution Load, Exposure Persistence, and Variability Severity — combined multiplicatively to capture compounding risk. WHO-normalized, confidence-weighted, and bounded to remain interpretable.

📡 The Data

CPCB monitoring data from 547 stations across India, fetched daily via the Government of India's open data API. Every snapshot is versioned in GitHub. The pipeline runs automatically every morning.

🌱 The Future

As data accumulates, AERSI scores become more statistically stable and meaningful. Long-term plans include seasonal trend analysis, state-level aggregations, and open API access for researchers.

How It Runs

The Data Pipeline

Fully automated. Runs every day at 10:30 AM IST without any manual intervention.

🌐

Fetch from CPCB API

Daily snapshot pulled from the Central Pollution Control Board via data.gov.in across all 547 active stations.

📦

Build 30-day rolling dataset

New snapshot merged with previous 29 days. Missing days detected and flagged. Confidence weight calculated.

🧮

Compute AQI and AERSI

CPCB sub-index AQI computed per station per day. Then PL, EPF, and VSF calculated and multiplied.

🗺️

Generate map and outputs

Interactive map rebuilt with fresh scores. All outputs committed back to GitHub automatically.

🌍

Site updates live on aersi.live

Vercel serves the latest committed data. No backend — the repository is the database.
